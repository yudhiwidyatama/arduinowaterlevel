# arduinowaterlevel
Pump / Water Level controlling with ultrasonic sensing JSN-SR04T and dotmatrix LCD display

# Input / Output
## Ultrasonic sensor : 
* JSN-SR04T Waterproof Ultrasonic Ranging Sensor, or
* hcsr04 hc sr04 ultrasonic distance sensor
## Relay:
* SSR SOLID STATE RELAY MODULE 2 CHANNEL 5V DC FOR ARDUINO

# Display
* OLED 1.3 inch 128X64 SPI LCD LED Display 4PIN - 0.91" => Adafruit SSD 1306  compatible, or
* OLED 1.3 inch 128X64 SPI LCD LED Display 4PIN - 0.96"


Compile using Arduino IDE (https://docs.arduino.cc/software/ide/)
